Game Boys Ranked By Sales

  1. Game Boy/Game Boy Pocket

    The Game Boy, Game Boy Pocket, and Game Boy Color sold a combined 118 million units. Nintendo combined these systems sales so its hard to tell the exact numbers but they'd sold 64 milion units when the Color was released so we'll attribute those to the Game Boy and Game Boy Pocket, which were very similar systems.

  2. Game Boy color

    We will attribute the other 54 million units to the Game Boy Color which puts it comfortably in third place.

  3. Game Boy Advance SP

    The Game Boy Advance SP sold over 43 million units thanks to its new clamshell design and very convenient backlight.

  4. Game Boy Advance

    Nintendo also combined units sold with it's "Advance" Game Boy line, however it was much easier to find the seperate sales numbers here. The Game Boy Advance is responsible for about 38 million of the 81 million in units sold combined with the SP.

  5. Game Boy Micro

    The Game Boy Micro sold just 2.5 million units and is known as one of the worst selling consoles of all time. despite this it's still a sought after device.
